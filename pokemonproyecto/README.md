# Pokedex

Bienvenido a la Pokedex, una aplicación web donde puedes explorar información sobre los Pokémon. Este proyecto fue realizado con el objetivo de aprender y aplicar conceptos de desarrollo web mientras se crea una interfaz interactiva y atractiva.

## Características

- **Búsqueda de Pokémon**: Encuentra información sobre cualquier Pokémon disponible en la base de datos.
- **Detalles de Pokémon**: Visualiza información detallada como nombre, tipo, estadísticas, movimientos, etc.
- **Interfaz amigable**: Diseño limpio y fácil de usar para una experiencia de usuario agradable.

## Tecnologías utilizadas

- **HTML5**: Estructura de la página.
- **CSS3**: Estilos y diseño visual.
- **JavaScript**: Lógica de la aplicación, manejo de eventos y manipulación de la interfaz.
- **API de Pokémon**: Fuente de datos externa para obtener información actualizada sobre los Pokémon.
